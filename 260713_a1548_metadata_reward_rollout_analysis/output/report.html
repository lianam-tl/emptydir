Proxy-gaming warnings

  • IoU explains more of the reward gain than value correctness.
  • Prediction-minus-GT segment count shifts upward by 2.95.
  • presence_type collapses toward upper body.
  • Movies/TV and soccer raw metadata decline slightly.

What was separated

  • Raw rule correctness: metadata value score before multiplying by segment IoU.
  • IoU-weighted score: core value × IoU term before JSON/schema eligibility. W&B metadata score additionally gates invalid outputs. It can rise because values improve, IoU improves, or both.
  • Conditional rule score: weighted score divided by available IoU credit; another value-quality view.
  • Semantic token F1: rough guardrail for metadata excluded from reward. It cannot replace human/LLM semantic evaluation.
